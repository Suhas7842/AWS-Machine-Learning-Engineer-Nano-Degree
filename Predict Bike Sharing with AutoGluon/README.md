# Predict Bike Sharing with AutoGluon

This project demonstrates automated regression modeling using **AutoGluon** to forecast daily bike rentals. The goal is to predict demand accurately by leveraging feature engineering, data preprocessing, and ensemble learning techniques.

## Problem Overview:

Bicycle sharing programs, also known as bike-sharing systems, offer a convenient and automated method for renting bicycles through a network of kiosk locations dispersed throughout urban areas. These systems enable individuals to rent a bike from one location and return it to another on an as-needed basis. At present, there are over 500 bike-sharing programs in operation worldwide.

The vast amount of data generated by these systems makes them an attractive subject for researchers as they provide explicit records of travel duration, departure and arrival location, and time elapsed. As such, bike-sharing systems serve as a sensor network, providing valuable insights into the mobility patterns of a city. This [Bike Sharing Demand Competition](https://www.kaggle.com/c/bike-sharing-demand) on Kaggle invites participants to utilize historical usage patterns and weather data to predict bike rental demand in the Washington D.C. based Capital Bikeshare program.

## Dataset:

Bike Sharing Demand. Kaggle. https://kaggle.com/competitions/bike-sharing-demand

## 📌 Project Overview

The project uses a publicly available bike-sharing dataset to:
- Explore daily rental patterns
- Identify predictive features
- Train and evaluate regression models using AutoGluon
- Generate accurate forecasts for bike rental demand

Key objectives include understanding feature importance, optimizing model performance, and producing reproducible results using AWS services.

## 🚀 Project Workflow

1. **Data Exploration & Cleaning**  
   - Examined dataset for missing values, outliers, and feature distributions  
   - Performed necessary transformations for model readiness

2. **Feature Engineering**  
   - Generated time-based features (day of week, month, holidays)  
   - Encoded categorical variables and normalized numerical features

3. **Model Training**  
   - Trained AutoGluon ensemble models for regression  
   - Performed hyperparameter optimization automatically

4. **Evaluation & Forecasting**  
   - Evaluated model performance using RMSE and MAE metrics  
   - Produced daily bike rental predictions with confidence intervals

5. **Results & Insights**  
   - Generated feature importance charts  
   - Visualized predicted vs. actual rental trends
