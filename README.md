# AWS Machine Learning Engineer Nanodegree Projects

This repository contains my completed projects from the **AWS Machine Learning Engineer Nanodegree**. The projects span multiple machine learning tasks—including regression, classification, workflow automation, and deep learning—using AWS services such as Amazon SageMaker.

---

## 📌 Overview

The AWS Machine Learning Engineer Nanodegree is a project‑based training program that teaches:
- Data preparation, EDA, and feature engineering
- Model training, deployment, and monitoring
- End‑to‑end ML workflows on AWS
- Deep learning for computer vision and real‑world use cases  
(Program outline is based on official course description.) :contentReference[oaicite:1]{index=1}

This repository organizes the projects completed during the nanodegree:
```
├── Predict Bike Sharing with AutoGluon
├── ML Workflow For Scones Unlimited On Amazon SageMaker
├── Image Classification using AWS SageMaker
├── Operationalizing an AWS ML Project
├── Capstone Project/ Heart Disease Prediction
└── README.md
```


---

## 📂 Project Summaries

### 📊 1. **Predict Bike Sharing with AutoGluon**
A demand forecasting model using AutoGluon to predict daily bike rentals. Includes:
- Data preprocessing & exploration
- Model training and evaluation
- Exported metrics and leaderboard submissions

---

### 🛠️ 2. **ML Workflow for Scones Unlimited**
Builds and deploys a complete machine learning pipeline using:
- Amazon SageMaker training and endpoints
- AWS Lambda for automation
- Step Functions for workflow orchestration

---

### 🐶 3. **Image Classification using AWS SageMaker**
Uses transfer learning to fine‑tune a CNN for classifying animal images:
- Preprocessing and dataset handling with SageMaker
- Training a deep learning model
- Deploying as a production endpoint

---

### 🚀 4. **Operationalizing an AWS ML Project**
Focuses on refining an ML model for production readiness:
- Monitoring and cost optimization
- Security best practices
- Redeployment improvements

---

### ❤️ 5. **Capstone: Heart Disease Prediction**
A final larger‑scale ML project using real patient data:
- Supervised learning approach
- Feature engineering and model tuning
- Prediction of disease risk
